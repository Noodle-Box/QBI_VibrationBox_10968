# Pip Installs

1. pyserial
2. sounddevice
3. numpy
4. depthai
5. opencv-python

Install all required Python packages with:

python -m pip install pyserial sounddevice numpy depthai opencv-python

Required non-pip dependency for FLAC audio export, H.265 video export, MP3 export, and audio/video merge:

- ffmpeg

Install ffmpeg on Windows with:

winget install Gyan.FFmpeg

Then restart PowerShell or VSCode terminal and verify with:

ffmpeg --version
